###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The code counts the number of truthy values in the nums array, which contains 0, 1, 2, 3. It uses a forEach loop to iterate through each element and increments the count variable when the element is truthy (not equal to 0). The final console.log count  statement outputs the count, which is 3, as there are three truthy values 1, 2, and 3 in the array.</i>

</p>
</details>
